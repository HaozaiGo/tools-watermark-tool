# 💧 Watermark Tool

> 在线图片水印工具 — 添加文字水印 · 保护版权 · 自定义样式

## ✨ 功能

| 功能 | 说明 |
|---|---|
| 🖼️ **图片上传** | 拖拽或选择图片，支持 JPG/PNG/WebP/GIF/BMP |
| ✏️ **文字水印** | 自定义水印文字内容、字体、大小、颜色 |
| 🎨 **样式控制** | 透明度、旋转角度、加粗、斜体、文字阴影 |
| 📍 **位置预设** | 9 个预设位置（四角/四边/居中），可调边距 |
| 🔁 **平铺模式** | 水印重复覆盖整张图片，防盗图效果更好 |
| 🔍 **实时预览** | 缩放预览，即时查看水印效果 |
| ⬇ **导出** | PNG（无损）或 JPG（可调质量）下载 |

## 🛠 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas API（水印渲染）
- 零外部依赖

## 🚀 部署

支持 Vercel / Netlify / Cloudflare Pages 等静态托管平台。

```bash
npx vercel --prod
```

## 🔒 隐私

所有处理在浏览器端完成，**图片不上传服务器**。
